## üìù Quick Reference

### MCP Methods
- `pyro_list_detonators` - List detonators
- `pyro_execute_detonator` - Execute detonator
- `pyro_create_case` - Create case
- `pyro_list_agents` - List agents
- `pyro_execute_pql` - Execute PQL
- `pyro_health` - Health check
- `pyro_authenticate` - Authenticate

### Fire Marshal Terms
- investigation (NOT hunt)
- detonator (NOT artifact)
- agent (NOT client)
- case (NOT session)
- collection (NOT execution)

### Configuration
- Config file: `pyro-detector-config.json`
- Environment: `PYRO_*` variables
- Cursor: `mcp-config.json`
