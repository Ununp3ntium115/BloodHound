# Steering Documents

This directory contains steering and reference documents for the Pyro project.

## Documents

### `cryptex-dictionary.md`
Complete dictionary of all functions and terms with anarchist branding. Use this for:
- Function name lookup
- Pseudocode reference
- Understanding call patterns
- Branding term mapping

### `project-workflow-a-to-b.md`
Complete guide on converting BloodHound Go codebase to Pyro Rust system. Includes:
- Phase-by-phase workflow
- Step-by-step instructions
- Tool usage examples
- Troubleshooting guide

## Usage

1. **Starting a new translation**: Read `project-workflow-a-to-b.md` Phase 1
2. **Looking up a function**: Check `cryptex-dictionary.md`
3. **Implementing in Rust**: Follow `project-workflow-a-to-b.md` Phase 3
4. **Troubleshooting**: Check both documents for solutions
